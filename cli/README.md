# myhippo

Copy-paste React UI components. Tailwind + motion. You own the code.

## Usage

Run in your project root.

```sh
npx myhippo clone                  # install everything
npx myhippo add button card        # install specific components
npx myhippo update button          # redownload (get latest)
npx myhippo remove button          # delete component file
```

Files land in `components/ui/` and helpers in `lib/`. No runtime dependency — edit freely.

## Peer deps

```sh
npm i motion lucide-react
```

Requires React 18+ and Tailwind CSS 4.
