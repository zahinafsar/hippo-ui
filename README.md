# hippo-ui

Copy-paste React components. Tailwind tokens, framer-motion animations, zero dependencies you don't want.

**Docs:** [zahinafsar.github.io/hippo-ui](https://zahinafsar.github.io/hippo-ui)

## Philosophy

- **Own the code.** Components live in your repo, not `node_modules`. Edit freely.
- **Tailwind for tokens, React for logic.** No CSS-in-JS, no variants DSL.
- **Animated by default.** `motion/react` for transitions that feel alive.
- **Minimal API.** No unnecessary subcomponents, no wrapper providers where a hook works.

## Getting started

```bash
git clone https://github.com/zahinafsar/hippo-ui
cd hippo-ui
npm install
npm run dev
```

Browse components at `http://localhost:3000/docs`, copy the source into your project, and go.

## Stack

- Next.js 16 (App Router, static export)
- React 19
- Tailwind CSS 4
- motion/react
- lucide-react
